# 导入组件

> 环境说明：

- React 16.8.0 及以上
- React DOM 16.8.0 及以上
- IM UI Kit 不支持 H5 界面搭建，仅支持 Web 浏览器界面搭建。

## 依赖安装

目前仅支持 NPM 方式集成，暂不支持 CDN 方式集成。

```bash
$ npm install @xkit-yx/im-kit-ui --save
```

在 React 项目中引入

```tsx
import {
  Provider, // 全局上下文
  ChatProvider, // 聊天组件上下文
  ConversationContainer, // 会话组件
  ChatContainer, // 聊天组件
  AddContainer, // 搜索——添加按钮组件
  SearchContainer, // 搜索——搜索组件
  ContactListContainer, // 通讯录——通讯录导航组件
  FriendListContainer, // 通讯录——好友列表组件
  GroupListContainer, // 通讯录——群组列表组件
  BlackListContainer, // 通讯录——黑名单列表组件
} from '@xkit-yx/im-kit-ui'

import '@xkit-yx/im-kit-ui/es/style'
```
