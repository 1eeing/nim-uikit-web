# 个人资料

UI Kit 提供了用于展示和编辑个人资料的组件 `MyAvatarContainer`。

`MyAvatarContainer` 有以下功能

- 默认展示个人头像
- 点击头像展开个人资料编辑页面，修改完可直接保存

## 代码示例

```tsx
import React from 'react'
import { Provider, MyAvatarContainer } from '@xkit-yx/im-kit-ui'

const App = () => {
  const initOptions = {
    appkey: '',
    account: '',
    token: '',
    debugLevel: 'debug',
    // ……
  }

  return (
    <Provider initOptions={initOptions} sdkVersion={2}>
      <div className="app">
        <div className="my-user-avatar">
          <MyAvatarContainer />
        </div>
      </div>
    </Provider>
  )
}
```
